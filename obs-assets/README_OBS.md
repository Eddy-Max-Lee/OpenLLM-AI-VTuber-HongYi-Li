# OBS Setup Notes

This MVP is an unofficial AI teaching assistant inspired by public ML teaching style. It is not Professor Hung-Yi Lee, not official, and not endorsed by him.

Recommended OBS setup:

1. Use Window Capture to capture the Open-LLM-VTuber window or the Chrome page at `http://localhost:12393`.
2. Add a Text source with this disclaimer:
   `Unofficial AI Tutor | Inspired by public ML teaching style | Not official`
3. Add `overlay_disclaimer.html` as a Browser Source if you want a reusable semi-transparent overlay.

Safety rules:

- Do not use Professor Hung-Yi Lee's photo as the avatar.
- Do not use cloned voice or voice conversion to imitate him.
- Do not title the stream as if it is an official stream or a stream by Professor Hung-Yi Lee.
- Keep the unofficial disclaimer visible in any public demo or stream.
